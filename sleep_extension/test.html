How to Test:

  1. Make sure the Keep Awake extension is installed and enabled
  2. Click the extension icon in the toolbar to cycle through states
  3. Watch for status messages at the top of this page
  4. Try leaving the page idle - the screen should stay on

  • System Mode: Both screen and system stay awake
  • Disabled: Normal power-saving behavior
